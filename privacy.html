Privacy Policy

1. はじめに

TablePlot – Event Layout Tool（以下「本アプリ」）は、イベントレイアウトを端末内で作成・編集し、必要に応じて画像として保存できるオフラインツールです。本ポリシーは、本アプリにおける情報の取扱いについて定めます。

最終更新日：

2. 収集する情報

本アプリは、以下の情報を収集しません。

  • 氏名、メールアドレス、電話番号などの個人情報
  • 位置情報
  • 広告ID
  • 解析/計測（Analytics）データ
  • 外部サーバーへの送信データ

3. 端末内で扱うデータ

  • 本アプリは、最後のレイアウト1件のみを端末内（ローカル）に保存します。
  • 保存先は端末のローカルストレージ（例：SharedPreferences等）であり、クラウド同期は行いません。

4. 画像保存（PNGエクスポート）について

  • ユーザー操作により、キャンバス全体をPNG画像として端末の写真ライブラリ（カメラロール等）へ保存できます。
  • 本アプリが画像を外部へ送信することはありません。

5. 権限について

画像保存のため、端末によっては写真ライブラリへのアクセス権限を求める場合があります。権限は画像保存機能のためにのみ使用されます。

6. 第三者提供

本アプリはユーザー情報を第三者に提供しません。

1. Introduction

TablePlot – Event Layout Tool (the “App”) is an offline tool to create/edit event layouts on-device and optionally save the canvas as an image. This policy explains how the App handles information.

Last updated:

2. Information We Collect

The App does not collect:

  • Personal information (name, email, phone number)
  • Location data
  • Advertising ID
  • Analytics/measurement data
  • Data transmitted to external servers

3. On-device Data

  • The App stores only the latest single layout locally on the device.
  • No cloud sync is performed.

4. PNG Export (Saving to Photo Library)

  • You can export the entire canvas as a PNG image and save it to your device’s photo library.
  • The App does not transmit images externally.

5. Permissions

Depending on your device/OS, the App may request permission to save images to your photo library. This permission is used only for the export feature.

6. Sharing with Third Parties

The App does not share user information with third parties.
