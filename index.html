アプリ概要（要点）

小規模イベント主催者向けの、オフライン完結型レイアウトツールです。指操作で机や区画を配置し、名前と色で視認性高く整理できます。
通信・アカウント・共有はありません。最後のレイアウト1件のみ端末内に保存し、必要に応じてキャンバスをPNGとして端末の写真ライブラリに保存できます。

App summary

An offline event layout tool for small organizers. Place tables/objects on a grid via touch, label and color-code items, store the latest layout locally, and export the canvas as a PNG to your photo library. No accounts, sharing, or networking.
